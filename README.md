<h1 id="english">MosaicsLi.github.io</h1>

This is the personal GitHub Pages website and frontend experiment repository for MosaicsLi. This project contains various Web development experiments, mini-games, UI/UX tests, and practice works using different frontend technologies (such as Vue, React, CSS, and API integrations).

## Project Contents & Git Tracked Items

Below is an overview of the main folders and file structure currently pushed to Git in the root directory:

### 📂 Projects & Experiments
- **`LabInterview/` & `LabInterview-dev/`**: Lab Visit Record Web Application (featuring Google Sheets API integration and medieval parchment style).
- **`ink-sort-puzzle/` & `ink-sort-puzzle-dev/`**: Ink Sort Puzzle mini-game.
- **`cocofolia-voice-extension/`**: Chrome extension experiment related to Cocofolia voice features.
- **`PhotoEditor/`**: A simple photo editor project.
- **`ProposalByChatGPT/`**: Projects proposed and generated with the help of ChatGPT (e.g., flashcard system).
- **`Schedule/`**: Calendar, scheduling, and data visualization experiments like D3.js.
- **`ListItme/`**: List and card UI tests (e.g., vocabulary cards, data lists).
- **`JustSelfIntriduce/`**: Self-introduction page design and effects (e.g., Glitch effect).
- **`HexoBlog/`**: Files related to the Hexo static blog.
- **`Calendar/`**: Calendar-related tests and implementations.
- **`ComboTest/`**: Web interaction or logic combo tests.

### 🧪 Frontend Tests
- **`JustTestSomeVue/`**: Vue.js framework practice and tests (including FFXIV related UI and data tests).
- **`JustTestSomeCSS.html` / `css/`**: Global experiments for CSS styles, animations, and layouts.
- **`JustTestSomeGrid.html`**: CSS Grid layout system tests.
- **`JustTestBootstrap.html`**: Bootstrap framework UI component tests.
- **`APItest.html`**: External API call and data presentation tests.

### 📄 Single Pages & Others
- **`index.html` / `index_old.html`**: Main entry page of the website and backup of the old homepage.
- **`ChessBoard.html`**: Chessboard UI rendering and logic tests.
- **`Justtest.html`**: Comprehensive miscellaneous web feature tests.
- **`tts.html`**: Text-to-Speech Web Speech API tests.
- **`WillCodeForFood.html`**: A fun/joke page test.
- **`MosaicsLi.github.io.code-workspace`**: VS Code workspace configuration file.
- **`.github/`**: GitHub related configuration files (like Actions).
- **`.gitignore`**: Git ignore rules.
- **`README.md`**: The documentation file for this project (this file).
